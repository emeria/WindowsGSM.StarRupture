# WindowsGSM.StarRupture

Prior to following these steps, forward port 7777 to your dedicated server's IP.

1. Add this folder to your plugins folder in WindowsGSM.
2. Create and start Server in WindowsGSM
3. Load up StarRupture on client
4. Choose 'Manage Server'
5. Enter your server IP
6. Add an admin password when prompted for password
7. Then click 'Change Password' and enter a password for players when they connect
8. Choose 'New Game' and enter a name, and click 'Start Game'
9. The server should now be running and you can go to the main menu again on any client and connect
